## How to Add a Skill

1. Build and test in `/home/moltbot/skillforge-bot/skills/`
2. Copy to `/home/moltbot/skill-library/<skill-name>/`
3. Update this INDEX.md
4. Deploy to target agent(s) if requested

## Skill Package Standard

Every skill must contain:
- `SKILL.md` — Main instructions
- `README.md` — What it does, who uses it
- `scripts/` — Automation (optional)
- `references/` — Reference docs (optional)
- `templates/` — Output templates (optional)
